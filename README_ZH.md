## ✨ 功能
- **超轻量级**：极低的电池消耗，仅在需要时唤醒。
- **直接启动**：无需裁剪界面，无需额外步骤。只需按住并搜索。
- **HD 触感反馈**：灵感来自 Pixel 设备的动态“震动”体验（1-100% 强度）。
- **隐私至上**：零数据收集。屏幕截图保留在本地并直接发送到 Google Lens。
- **完全可定制**：调整检测区域、位置和振动强度。
- **多语言**：支持 EN、FR、DE、NL 和 ZH。

## ⚙️ 系统要求
- Android 11 (API 30) 或更高版本。
- 已安装 Google 应用（用于 Google Lens）。
- 已授予无障碍服务权限。
