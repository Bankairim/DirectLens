# DirectLens 🔍

[![License](https://img.shields.io/badge/License-GPLv3-blue.svg)](LICENSE)
[![GitHub release](https://img.shields.io/github/v/release/Bankairim/DirectLens)](https://github.com/Bankairim/DirectLens/releases)

**DirectLens** ist eine ultraleichte Open-Source-Alternative zu „Circle to Search“ für jedes Android-Smartphone ab Android 11+. 

Starten Sie Google Lens sofort durch langes Drücken Ihrer Navigationsleiste, mit einem erstklassigen haptischen „Pixel-ähnlichen“ Erlebnis.

## ✨ Funktionen
- **Ultraleicht**: Minimaler Batterieverbrauch, wird nur bei Bedarf aktiviert.
- **Direkter Start**: Keine Zuschneideoberfläche, keine zusätzlichen Schritte. Einfach drücken und suchen.
- **HD Haptisches Feedback**: Dynamisches „Buzz“-Erlebnis (1–100 % Intensität), inspiriert von Pixel-Geräten.
- **Datenschutz an erster Stelle**: Keine Datenerfassung. Screenshots bleiben lokal und gehen direkt an Google Lens.
- **Vollständig anpassbar**: Erkennungszonen, Position und Vibrationsstärke anpassen.
- **Mehrsprachig**: Unterstützt EN, FR, DE, NL und ZH.

## ⚙️ Voraussetzungen
- Android 11 (API 30) oder höher.
- Google App installiert (für Google Lens).
- Berechtigung für den Bedienungshilfen-Dienst erteilt.
